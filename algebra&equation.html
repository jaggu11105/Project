Algebra & Equations

1. Combining Like Terms

  • 3x + 2x = 5x

2. Distributive Property

  • a(b + c) = ab + ac

3. Factoring

  • ax + ay = a(x + y)

4. Slope-Intercept Form of a Line

  • y = mx + b

5. Point-Slope Form of a Line

  • y - y_1 = m(x - x_1)

6. Standard Form of a Line

  • Ax + By = C

7. Solving for a Variable

  • Isolate the variable on one side of the equation

8. Standard Form of a Quadratic Equation

  • ax^2 + bx + c = 0

9. Quadratic Formula

  • x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}

10. Vertex Form of a Quadratic Equation

  • y = a(x - h)^2 + k

11. Factoring Quadratic Equations

  • ax^2 + bx + c = (px + q)(rx + s)

12. Adding/Subtracting Polynomials

  • Combine like terms

13. Multiplying Polynomials

  • Use the distributive property

14. Dividing Polynomials

  • Use polynomial long division or synthetic division

15. Factoring Polynomials

  • Factor out the greatest common factor, use the difference of squares, or factor by grouping

16. Solving Linear Inequalities

  • Solve similarly to linear equations, but reverse the inequality sign when multiplying or dividing by a negative number

17. Solving Quadratic Inequalities

  • Solve the corresponding quadratic equation to find critical points, then test intervals

18. Solving Absolute Value Inequalities

  • Solve by considering the definition of absolute value

19. Distance Formula

  • d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}

20. Midpoint Formula

  • \left( \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2} \right)
