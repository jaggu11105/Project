Data Interpretation

1. Percentage Calculation

  • Percentage = \left( \frac{\text{Part}}{\text{Whole}} \right) \times 100

2. Percentage Increase/Decrease

  • Percentage Change = \left( \frac{\text{New Value} - \text{Old Value}}{\text{Old Value}} \right) \times 100

3. Average

  • Average = \frac{\text{Sum of all values}}{\text{Number of values}}

4. Weighted Average

  • Weighted Average = \frac{\sum (\text{Value} \times \text{Weight})}{\sum \text{Weight}}

5. Mean

  • Mean = \frac{\sum \text{Values}}{\text{Number of values}}

6. Median

  • Odd number of values: Middle value
  • Even number of values: Average of the two middle values

7. Mode

  • The value that appears most frequently in a data set.

8. Range

  • Range = \text{Maximum Value} - \text{Minimum Value}

9. Standard Deviation

  • \(\sigma = \sqrt{\frac{\sum (x_i - \mu)^2}{N}}\)
  • \(x_i\) = each value
  • \(\mu\) = mean
  • \(N\) = number of values

10. Variance

  • Variance = \(\sigma^2 = \frac{\sum (x_i - \mu)^2}{N}\)

11. Quartiles

  • First Quartile (Q1): 25th percentile
  • Second Quartile (Q2): 50th percentile (Median)
  • Third Quartile (Q3): 75th percentile

12. Interquartile Range (IQR)

  • IQR = Q3 - Q1

13. Z-Score

  • Z = \frac{(X - \mu)}{\sigma}
  • X = value
  • \(\mu\) = mean
  • \(\sigma\) = standard deviation

14. Correlation Coefficient

  • r = \frac{n(\sum xy) - (\sum x)(\sum y)}{\sqrt{[n\sum x^2 - (\sum x)^2][n\sum y^2 - (\sum y)^2]}}
  • n = number of observations
  • x, y = values

15. Regression Line

  • y = a + bx
  • a = intercept
  • b = slope

16. Slope of a Line

  • b = \frac{n(\sum xy) - (\sum x)(\sum y)}{n(\sum x^2) - (\sum x)^2}

17. Intercept of a Line

  • a = \frac{(\sum y)(\sum x^2) - (\sum x)(\sum xy)}{n(\sum x^2) - (\sum x)^2}

18. Compound Annual Growth Rate (CAGR)

  • CAGR = \left( \frac{\text{Ending Value}}{\text{Starting Value}} \right)^{\frac{1}{n}} - 1
  • n = number of years

19. Moving Average

  • Moving Average = \frac{\sum \text{Values in the period}}{\text{Number of periods}}

20. Exponential Smoothing

  • S_t = \alpha \times Y_t + (1 - \alpha) \times S_{t-1}
  • S_t = smoothed statistic at time t
  • Y_t = actual value at time t
  • \(\alpha\) = smoothing factor (0 < \(\alpha\) < 1)
